#  RPG Manage

RPG Manage é um projeto desenvolvido em JavaScript com o objetivo de praticar lógica de programação através da criação de um sistema simples de gerenciamento de personagens de RPG.

##  Funcionalidades

- Criar personagens
- Escolher classe (Guerreiro, Assassino, Mago ou Arqueiro)
- Visualizar personagens cadastrados
- Treinar personagens e ganhar experiência
- Sistema de nível (Level Up)
- Sistema de batalhas contra inimigos
- Batalhas contra chefes (Boss Battle)
- Loja para compra de equipamentos
- Inventário
- Equipar armas e armaduras
- Excluir personagens

##  Tecnologias utilizadas

- JavaScript
- Arrays
- Objetos
- Funções
- Estruturas de repetição (`for`)
- Estruturas condicionais (`if`, `switch`)
- Métodos de Arrays (`push`, `splice`, `indexOf`)
- `Math.max()`

##  Objetivo

Este projeto foi desenvolvido como parte dos meus estudos em JavaScript, com foco em praticar lógica de programação, manipulação de objetos e criação de sistemas utilizando apenas JavaScript.
